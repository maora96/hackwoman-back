# hackwoman-back

back-end for the project developed during the provi hack woman hackathon

## technologies

- nodejs
- postresql
- koa

## dependencies

- koa
- koa-bodyparser
- koa-router
- @koa/cors
- dotenv
- pg
- jsonwebtoken
- bcryptjs

## start project

```bash
# clone repository
$ git clone https://github.com/maora96/hackwoman-back.git

# install Dependencies
$ npm install

# run app
$ npm start
```
